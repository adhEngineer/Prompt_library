# Prompt Library

Bibliotecă de prompturi organizată pe concepte reutilizabile.

## Concepte

1. **Concept 1** – șablon simplu (placeholder).
2. **Concept 2** – șablon structurat pentru cereri clare, constrângeri explicite și format de output predictibil.

## Concept 2

Vezi implementarea completă în: [`prompts/concept-2.md`](prompts/concept-2.md).

Scopul conceptului 2 este să reducă ambiguitatea prin:
- context inițial clar,
- obiectiv măsurabil,
- reguli explicite,
- format standard de răspuns.

## App (MVP)

Am început implementarea aplicației pentru Concept 2 în `app/`.

### Rulează local

Poți deschide direct `app/index.html` în browser sau porni un server static:

```bash
python3 -m http.server 8000
```

Apoi accesează `http://localhost:8000/app/`.
